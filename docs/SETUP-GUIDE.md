## Overview

This guide provides step-by-step instructions for deploying the AI-Powered Phishing Detection System on your local machine. Total setup time: approximately 15-20 minutes.

## Prerequisites

### Required Software
- **Python 3.9 or higher** (I used Python 3.12.6)
- **Git** for cloning repositories
- **Visual Studio Code** (or any code editor)
- **4GB RAM minimum** (8GB recommended)

### Operating System
- Windows 10/11
- macOS
- Linux (Ubuntu, Debian, etc.)

### Skill Requirements
- Basic command line navigation
- Familiarity with Python and pip
- Willingness to troubleshoot!

---

## Installation Steps

### Step 1: Create Project Workspace

Open your terminal and create a dedicated folder:

**Windows:**
```bash
cd Desktop
mkdir CSN190
cd CSN190
```

**Mac/Linux:**
```bash
cd ~/Desktop
mkdir CSN190
cd CSN190
```

---

### Step 2: Clone the Repository
```bash
git clone https://github.com/GauravGhandat-23/AI-Powered-Phishing-Detection-System
```
